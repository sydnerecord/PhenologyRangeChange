3 - Bayesian Regression Model Scripts

01_Prep_Regression_Data: This script merges the ensemble projection differences data for phenology and range in preparation for incorporating these SDM outputs into subsequent Bayesian Regression Models to test hypotheses 1 and 2.

02_MC_correlation_check: This script ensures that the predictor variables included in the Bayesian Regression Models meet the assumptions of non-collinearity.

03_Run_Bayesian_Regression_Models: Runs the Bayesian Generalized Linear Models (i.e., regression models) to test hypotheses 1 and 2.

04_Plots: Plots the outputs of the Bayesian regression models.

05_Tables: Generates table outputs of the Bayesian regression model coefficients.
